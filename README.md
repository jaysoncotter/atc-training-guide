# ATC Transcription Training Guide

An interactive, browser-based training guide designed to help new 
contributors learn Air Traffic Control transcription formatting 
standards and pass their qualification evaluation.

## Features

- **Flip-card flashcards** for active recall practice
- **Progressive practice exercises** with reveal-answer functionality
- **Side-by-side formatting comparisons** (unformatted vs. correctly formatted)
- **Annotated real-world ATC communication patterns** covering the 20 most common exchanges
- **Reference tables** for callsigns, waypoints, phonetics, and formatting rules
- **External resource links** (SkyVector, AirNav, FlightAware) for verification workflows

## Background

I built this independently after identifying that new team members 
were struggling to absorb complex ATC transcription guidelines. 
The guide restructures dense documentation into concise, testable 
Q&A pairs optimized for comprehension and retention.

It is now used as an onboarding resource for contributors preparing to pass the ATC qualification evaluation.

## Tech

Single-page HTML/CSS/JavaScript — no frameworks, no dependencies. 
Just open `index.html` in any browser.
